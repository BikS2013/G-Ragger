## seed request

I want to use Gemini File Search to create workspaces where I can upload my files for indexing and search them
when I need to answer questions.
I want a tool that allows me to create new spaces and upload files.
Sometimes I am gonna ask to upload content from disk, or directly from the web, either from a web page, or from a
YouTube video, and sometimes I am gonna ask to add just a personal note on a workspace.

Each upload I add to a workspace must have a timestamp and a title, which might be the web page title, the YouTube
 video title, or, for personal notes, a title based on the note content.
Timestamps and titles must be generated directly by the tool.
The tool must allow me to change the title for the uploads.
The tool must allow me to remove an upload.
I want the option to add an expiration date to uploads. I want the option to add completed, urgent, or inactive
flags.
I want the tool to provide a list of labels for metadata properties used in a workspace.
I want the tool to support using metadata as filters when I ask the workspace questions.
